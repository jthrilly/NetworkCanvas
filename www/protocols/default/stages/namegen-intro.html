Name generators in Network Canvas

Perhaps the most fundamental step in network interviews is having ego nominate other people ("alters") based on some criteria. This step is referred to as a "name generator".

Network Canvas can integrate many types of data into a name generator. For example, the interview designer can use a roster, or names nominated from a previous interview.

In this name generator, we import two kinds of data to be used as alter attributes. The first is a list of roles (displayed on the next screen as [set relationship roles]) and the second is a shortened list of groups based on the responses to the previous page.

On the following screen, use the "add node" button in the bottom right to create a new alter.
