Name generators in Network Canvas

Name generators are a fundamental activity to be undertaken in almost all network interviews.

In Network Canvas, name generators inherit the flexibility afforded by digital approaches. Data from external sources (rosters, previous interviews, CSV fies) can be brought in, and participants can edit any data they have provided seamlessly.

Here, we use data collected in the previous step as an attribute of Ego, and create a custom control in the 'new alter' form.
