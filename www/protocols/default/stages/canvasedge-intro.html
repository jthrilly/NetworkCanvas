Creating links between alters

As mentioned on the previous screen, Network Canvas supports many different types of task, all aided by the visual 'framework' provided by the sociogram.

Here, we present an interface for creating edges or links between alters. Network Canvas supports the creation of multiple types of links, which can be selectively shown or hidden from future sociogram interfaces.

On the next screen you can draw a line between people by clicking one node and then clicking a second node.
